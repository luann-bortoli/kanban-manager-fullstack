## Kanban Board – React + Flask
<img width="1901" height="905" alt="Screenshot_2" src="https://github.com/user-attachments/assets/4e747ee5-1e2a-4d67-a77f-cb732353d58c" />

Aplicação fullstack que implementa um Kanban Board utilizando ReactJS (Vite) no frontend e Python com Flask no backend.
As tarefas são persistidas em um arquivo JSON, utilizado como banco de dados.

# Funcionalidades:
  - Visualizar título, descrição, e prazo das taks.
  - Criar novas tarefas
  - Mover tarefas entre colunas (Todo, WIP e Done)
    
# Tecnologias usadas
  - ReactJS
  - Vite
  - Python + Flask
